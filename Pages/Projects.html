Green Screen

In this Project I use java to make a Green Screen. It takes in two images the first is the one that you want to be your green screen and the other is the image you want over it, then I ask the user to give me color ranges for red,blue and green and then the program looks at each pixel and asses if the pixel falls with in the range to be over written if it does then the program will overwrite the current pixel on the green screen image with the other image
